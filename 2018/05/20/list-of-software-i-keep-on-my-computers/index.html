List of Software I Keep on My Computers

2018-05-20 · 76 words · 1 minute read
new computer · software

In the event that I have to replace a computer, these are the software packages that I get (in the order I need them)

  • Google Chrome
  • Sublime text
  • Pycharm
  • Evernote
  • Git (and Gitbash if I’m on windows)
  • 7zip
  • Visual studio code
  • Krita
  • Openshot
  • Audacity
  • OBS
  • Handbrake
  • FFMPEG
  • Youtube-dl
  • Spotify
  • Docker
  • WinDirStat (grand perspecive on OSX)

For Video Game Work

  • Blender
  • Unity3D
  • Zbrush 4 (If I need to do some serious 3D work)
  • Razer Cortex (Windows only)
